易经

十天干十二地支

甲、乙、丙、丁、戊、己、庚、辛、壬、癸被称为“十大天干” ，其中甲丙戊庚壬为阳，乙丁己辛癸为阴。
子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥叫作“十二地支”，子寅辰午申戌为阳，丑卯巳未酉亥为阴。

十二时辰

子时 ( 23时至01时) 丑时 ( 01时至03时) 寅时 ( 03时至05时) 卯时 ( 05时至07时) 辰时 ( 07时至09时) 巳时 ( 09 时至11时) 午时 ( 11时至13时) 未时 ( 13时至15时) 申时 ( 15时至17时) 酉时 ( 17时至19时) 戌时 ( 19时至21时) 亥时 ( 21时至23时)

两仪

  1. 易经由卦组成， 卦由爻组成爻，分为阴爻（–）、阳爻（一）又叫两仪。
  2. 在秦朝使用一 和 六 代表阳爻和阴爻(其中一也等于七)。
  3. 除了六和七代表， 八和九也很重要， 其中九代表阳爻， 第一个阳爻表示初九，第二个叫九二，依次内推（七和九代表阳爻，六和八是阴，偶数为阴奇数为阳）；

阴阳

  1. 一到十之中奇数为阳，偶数为阴， 也要阳数和阴数或者天数和地数。
  2. 但是只取其中的六七八九来成为爻的阴阳。

生数&成数

天一生水，地二生火，天三生木，地四生金。地六成水，天七成火，地八成木，天九成金，天五生土。（《尚书大传·五行传》）

  1. 一二三四五 为生数，五六七八九十为成数。
  2. 水火木金土分别对饮五行生和成的状态。
  3. 为什么只取4个数字成为爻的阴阳？原因是一到五这几个数字为生数或者说是先天先验的是一种气（即物质还没形成之前的趋势）不是实实在在的物质。后面的成数才代表物质是为后天的经验的，但是其中的十代表土，古代认为土不分阴阳属于阴阳的融合状态。故只有4个数字成为爻的阴阳。

四象

太极生两仪，两仪生四象，四象生八卦。
左青龙，右白虎，北玄武，南朱雀。

  1. 四象即为六（水-玄武）七（火-朱雀）八（木-苍龙）九（金-白虎）, 对应四种圣兽(东宫苍龙，南宫朱雀，西宫白虎，北宫玄武)。

苍龙（木）：
春在东天（木）喜乐，夏在南天（火）助火，
秋在西天（金）受制，冬在北天（水）得生。
朱雀（火）：
春在南天（火）喜乐，夏在西天（金）制金，
秋在北天（水）受制，冬在东天（木）得生。
白虎（金）：
春在西天（金）喜乐，夏在北天（水）助水，
秋在东天（木）制木，冬在南天（火）受制。
玄武（水）：
春在北天（水）喜乐，夏在东天（木）助木，
秋在南天（火）制火，冬在西天（金）得生。

五行
金木水火土：木生火，火生土，土生金，金生水，水生木； 金克木，木克土，土克水，水克火，火克金；

对人五脏对应：木-肝胆，火-心小肠，土-脾胃，金-肺大肠，水-肾膀胱

河图

  1. 分别为水火木金土 的生数和成数
  2. 来源：古代人利用天体的移动和变化规律。

新宿二

中国古代星占家认为火星在位置及亮度上都常变不定，谓之“荧惑”，象征残、疾、丧、饥、兵等恶象。而心宿内的心宿一、心宿二和心宿三因与紫微垣中的北极五星构造相似，被认为分别对应太子、帝和庶子。因此荧惑在心宿中由运行方向的变易常常被认为是“大人易政，主去其宫”的征兆。[1] 例如：西汉成帝绥和二年（前7年），天官报告荧惑守心，宰相翟方进被汉成帝赐毒酒自杀。翟方进死后几天，汉成帝突然暴毙，后来王莽居摄，翟方进之子翟义起兵反王莽。 上一次荧惑守心发生于2016年，下一次荧惑守心将发生于2048年。(维基百科)

注：在中国古代叫商星（参星和商星，彼此为天空的两端，参星和商星永不相见），也叫大火（对应天蝎座）。

二分二至

寅宾出日，平秩东作。日中，星鸟，以殷仲春——尧典 （表示春分）
平秩南讹，敬致。日永，星火，以正仲夏——尧典（表示夏至）
寅饯纳日，平秩西成。宵中，星虚，以殷仲秋——尧典（表示秋分）
平在朔易。日短，星昴，以正仲冬。——尧典（表示冬至）

八卦

说明：
八卦分别为：
北方为坎卦
东方为震卦
南方为离卦
西方为兑卦





在宋朝以后有分为先天八卦和后天八卦， 但是之前没有先天后天之分，故这个可以忽略；

1.八卦与地支：空间即指代八卦， 时间指代十二地支（或者说十二个月份，地支和月份是对齐的）；
2.八卦一共分为八个方向， 4个正方向（东西南北），4个偏方向；
